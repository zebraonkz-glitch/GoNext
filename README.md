# GoNext

Помощник туриста — офлайн-дневник мест, поездок и попутчиков (Expo SDK 57).

## Запуск

```powershell
npm install
npm start
```

Откройте приложение в Expo Go (Android/iOS) или эмуляторе.

## Проверка перед сборкой

```powershell
npm run typecheck
npm run doctor
```

## Сборка (нативное приложение)

Требуется Android Studio (Android) или Xcode на macOS (iOS).

```powershell
# Android — development build
npm run prebuild:android
npm run run:android

# iOS (только macOS)
npm run prebuild:ios
npm run run:ios
```
